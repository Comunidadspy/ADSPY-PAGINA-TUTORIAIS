Backup de pixel:


1 perfil e 1 bm apenas de backup em 1 navegador criado no Incogniton


Crie o pixel e compartilhe com as bms anunciantes


De mês em mês faça 1 campanha de envolvimento no valor mínimo e desative(apenas para a bm não ser desativada por falta de uso). Não use para anunciar

Navegadores anunciantes:


Os outros navegadores você vai usar para anunciar. Com o pixel compartilhado da bm backup os riscos de perder o pixel são menores

Como usar o Incogniton:


Com a conta trial que você criou no site e com o Incogniton instalado e aberto, acesse com o email/usuário e senha


Clique em CREATE SINGLE (menu esquerdo Browser Profile Creation)


Em "Browser profile name" escolha um nome para o navegador, recomendo colocar uma descrição, por exemplo (bm backup de pixel)


Operating system deixe Windows


Para utilizar proxy, acesse o menu Proxy (menu esquerdo), selecione o tipo de proxy (se estiver usando o Blazing escolha HTTP).

Adicione a proxy no formato ip:porta e o username e password da proxy


Clique em Check Proxy para ver se está funcionando, e para criar o navegador clique em Create Profile


Se não abrir o navegador, procure no Incogniton e clique em START
